# OMO

오마카세 모여라!
디프만 10기 8조 프론트엔드 저장소입니다.

### 개발 모드

```shell
$ yarn dev
```

로컬 컴퓨터에서 개발 모드로 프로젝트를 실행하면, 수정한 파일을 저장했을 때 코드 변경 사항이 자동으로 브라우저에 반영됩니다.

### 배포 모드

```shell
$ yarn build
$ yarn start
```

코드 변경 사항이 자동으로 반영되진 않지만 코드 최적화로 인해 실행 속도가 빠릅니다.

### 브라우저 실행

```
http://localhost:3000
```

브라우저에서 아래 주소로 접속하면 개발 중인 사이트를 볼 수 있습니다.
